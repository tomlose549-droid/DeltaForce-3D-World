# 游戏完整设计思路

> 最后更新：2026-05-23（V1.8）

---

## 一、游戏概述

这是一款基于 **Three.js** 构建的 **第一人称 3D 探索游戏**，运行于浏览器，无需安装。玩家置身于一个封闭的大理石围合空间中，通过与场景中的交互元素触发卡牌系统界面，整体风格融合了写实场景与霓虹赛博朋克 UI 的视觉对比。

---

## 二、世界场景设计

### 地面
- 无限延伸的网格地板，便于玩家感知移动距离与方向感

### 围墙
- 以玩家出生点为中心，内侧 20×20 格的正方形封闭空间
- 四面大理石贴图墙壁，高 3 格，顶部开放
- 大理石采用真实无缝贴图（marble.jpg），经 PBR 材质渲染，具有轻微光泽质感
- 墙壁有碰撞检测，玩家无法穿越

### 天空
- 淡灰蓝色天空背景，带轻微指数雾效增强纵深感
- 高空（约 55 格高）漂浮 12 个半透明黑色玻璃立方体
- 每个方块大小随机（20~28 格），带蓝色边框线条
- 持续缓慢漂移 + 自转，营造神秘漂浮感

### 光照
- 环境光（均匀补光）+ 方向光（模拟太阳）+ 半球光（天空漫反射补光）
- 三光源组合确保场景明亮自然，大理石暗部细节清晰可见

## 三、玩家控制

| 操作 | 功能 |
|---|---|
| WASD | 前后左右移动 |
| Shift | 加速移动（3倍速） |
| 鼠标移动 | 转动视角（指针锁定模式） |
| ESC | 退出游戏，返回设置页 |
| 空格 | 与准星对准的物体交互 |

进入设置页提供视角灵敏度滑块（0.2x ~ 3.0x），可在进入前调整，ESC 退出后也可重新调整。

---

## 四、视觉效果

### 准星
- 白色十字线固定在屏幕正中央，始终可见

### 移动粒子
- 玩家移动时，脚下随机生成 10~30 个白色圆形粒子
- 粒子向上飘散，持续 2 秒后消失
- 停止移动后，已生成粒子继续飘散直至消失
- 采用自定义 ShaderMaterial 实现逐粒子透明度

### 卡牌按钮
- 贴于北墙内面，视线高度
- 黑底卡牌，边框与"?"文字均为彩虹循环变色霓虹辉光
- 通过 ShaderMaterial + 强度图实现实时动态彩虹效果

---

## 五、音效设计

- 模拟室内硬地板（大理石 / 瓷砖）的清脆撞击感
- Web Audio API 合成，无需外部音频文件
- 左右脚交替播放，两声略有频段差异（1900Hz / 2300Hz）
- 带 30ms 短延迟模拟室内硬墙回响
- 普通行走：1.5 步/秒；Shift 加速：3 步/秒

## 六、交互系统

### 准星检测
- 玩家准星对准卡牌（5格范围内）时，屏幕右下角出现"按[空格]交互"提示
- 按空格触发 2D 卡牌界面，同时自动解除鼠标锁定

### 2D 卡牌界面
- 全屏覆盖，黑色背景 + 彩虹流动渐变纹路 + 斜线装饰
- 所有按钮统一风格：黑底 + 彩虹渐变动态边框

### 顶部导航逻辑

| 状态 | 显示按钮 | 中央内容 |
|---|---|---|
| 默认 | ✕ / 抽卡 / 容器 | 空 |
| 点击抽卡 | ✕ / 容器 | 2列网格，6个地点按钮 |
| 点击容器 | ✕ / 抽卡 | 横排，3个分类按钮 |

### 抽卡子面板（6个地点）

| 列1 | 列2 |
|---|---|
| 常规零号大坝 | 机密巴克十 |
| 机密航天基地 | 绝密巴克十 |
| 绝密航天基地 | 绝密潮汐监狱 |

点击地点按钮后进入集装箱开箱面板：
- 各地点概率随机生成 2～6 个灰黑条纹集装箱
- 点击集装箱：按战利品表抽取物品，显示 Canvas 图标 + 名称
- ✓ 确认收取返回地点选择

## 七、技术架构

| 模块 | 技术方案 |
|---|---|
| 3D 渲染 | Three.js r160（ES Module CDN） |
| 视角控制 | PointerLockControls |
| 材质 | MeshStandardMaterial（PBR）+ ShaderMaterial |
| 音效 | Web Audio API 合成 |
| 粒子 | 自定义 GLSL Shader |
| 版本管理 | Git（本地仓库，master + 临时分支） |
| 本地预览 | Python HTTP Server（端口 8080，开机自启） |
